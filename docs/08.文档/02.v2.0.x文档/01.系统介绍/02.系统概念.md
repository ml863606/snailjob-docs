# ⚡组 (group)
> 组不支持自动创建，在使用前必须在控制台手动创建一个组

组是一个逻辑概念, 它可以看做是场景的分类或者是标签, 用于将一些相关性的场景进行分组，可以理解为MQ的Topic和Queue的关系; 服务端的负载均衡也是基于组
把不同的组分配到不同的服务端集群的其中一个节点。

# ⚡场景(scene)
> 在配置组时选择自动创建场景，则系统发现存在场景时会自动创建一个场景

每个组对应N个场景且同一个组不能有相同的场景，场景是系统配置的最小单位
