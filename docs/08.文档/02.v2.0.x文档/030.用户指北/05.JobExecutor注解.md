## JobExecutor注解全貌

| 参数                  | 描述      | 默认值                   | 必须指定 |
| --------------------- |---------| --------------------- | -------- |
| name                 | 执行器名称   | 无                     | ✅        |
| method               | 执行器方法名称 | jobExecute            | ❌        |

执行器名称: 指任务的标识，同一个客户端不能有想同的执行器名称
执行器方法名称: 执行任务的方法名称
